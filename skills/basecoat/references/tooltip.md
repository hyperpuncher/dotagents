# Tooltip

```html
<button data-tooltip="Tooltip text">Hover me</button>
<button data-tooltip="Text" data-side="right">Right</button>
<button data-tooltip="Text" data-side="bottom" data-align="start">Bottom start</button>
```

Attrs: `data-tooltip`, `data-side` (top/bottom/left/right), `data-align` (start/center/end)
