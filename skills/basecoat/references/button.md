# Button

Classes: `btn` (primary), `btn-secondary`, `btn-destructive`, `btn-outline`, `btn-ghost`, `btn-link`, `btn-icon`

Combine: `btn-sm-icon-destructive`, `btn-lg-outline`

```html
<button class="btn">Button</button>
<button class="btn-icon-outline"><!-- icon --></button>
<button disabled><svg class="animate-spin"></svg> Loading</button>
```

# Button Group

```html
<div role="group" class="button-group" data-orientation="vertical">
  <button class="btn-outline">A</button>
  <hr role="separator" />
  <button class="btn-outline">B</button>
</div>
```
